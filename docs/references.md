# References and acknowledgements

The SSCC 2026 Quantum Chemistry hands-on sessions make use of the
following open-source software packages. We gratefully acknowledge
the work of their developers.

---

### ORCA

Neese, F.; Wennmohs, F.; Becker, U.; Riplinger, C.
The ORCA quantum chemistry program package.
*J. Chem. Phys.* **2020**, *152*, 224108.
[https://doi.org/10.1063/5.0004608](https://doi.org/10.1063/5.0004608)

👉 [ORCA forum](https://orcaforum.kofo.mpg.de) · [ORCA 6 tutorials](https://www.faccts.de/docs/orca/6.0/tutorials/)

!!! note
    ORCA prints a list of recommended method-specific citations at the
    end of every output file. Please cite these in addition to the
    generic reference above when publishing results.
