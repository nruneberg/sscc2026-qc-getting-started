# Working environment

The SSCC 2026 Quantum Chemistry hands‑on uses
**CSC Jupyter for Courses**.

All hands‑on work is carried out in this environment.
No local installation or software setup is required.

## Access

- Log in using your CSC or Haka account
- Ensure multi‑factor authentication (MFA) is activated

## During the session

1. Open the SSCC 2026 QC course in CSC Jupyter for Courses
2. Launch the provided notebook environment
3. Open the notebooks indicated during the session

All required software, including **ORCA** and **CP2K**
(where applicable), is preconfigured in the environment.
